# The Peter Mageto Leadership Portfolio

Fresh executive client-site starter for a professional leadership portfolio for Rev. Professor Peter Mageto, fifth Vice Chancellor of Africa University.

For full handoff notes, roadmap, architecture, quality checks, and future enhancements, read [`PROJECT_DOCUMENTATION.md`](./PROJECT_DOCUMENTATION.md).

## Architecture

- `src/App.jsx` contains the page structure and verified portfolio content.
- `src/styles.css` contains the complete visual system: responsive layout, executive color palette, typography, cards, timeline, and contact sections.
- `index.html` contains SEO and social metadata.
- The app is Vite + React so it is fast to run, easy to host, and simple for a mentor or developer to review.

## Run

```bash
npm install
npm run dev
```

Build and preview production output:

```bash
npm run build
npm run preview
```
